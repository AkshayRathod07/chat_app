# chat_app_digi_alpha

TypeScript + Express + MongoDB minimal scaffold.

Quick start

1. Copy `.env.example` to `.env` and set `MONGO_URI`.

2. Install dependencies:

```bash
npm install
```

3. Start development server (uses `nodemon` + `ts-node`):

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
npm start
```
